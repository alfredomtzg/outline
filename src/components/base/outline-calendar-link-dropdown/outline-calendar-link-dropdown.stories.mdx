## Calendar Link Dropdown Attributes

- `title` string (required) - The event title. 
- `start` string (required) - The event start time. Accepted values formats are JS Date | ISO 8601 string | Unix Timestamp
- (*)`end` string - The event end time. Accepted values formats are JS Date | ISO 8601 string | Unix Timestamp
- (*)`duration` [number, string] - The event duration. Format is an array with value(number) and unit(string) i.e [3, "hour"]. See allowed unit types [here](https://day.js.org/docs/en/durations/creating#list-of-all-available-units)
- (*)`all-day` boolean - Whether this is an all day event. 
- `description` string - Event description
- `location` string - Event location
- `busy` boolean - Whether to mark on calendar as busy
- `guests` string[] - Emails of guests i.e ['user1@test.com', 'user2@test.com']
- `url` string - Calendar document URL

(*) You must provide either `end`, `duration`, or `all-day`.
